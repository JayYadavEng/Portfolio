Personal Projects

Desktop Display

  • Small, desktop screen that displays information like the weather, currently playing songs, and time
  • Inspiration from the Tidbyt, and the belief I could make a much cheaper clone than its $200 asking price
  • Coded with MicroPython and built with a Raspberry Pi and Matrix Display
  • Case designed in SOLIDWORKS and manufactured from 3D printing

Spotify Mini-Player

  • Coded a miniplayer that features the currently playing song and interactive control buttons in MicroPython
  • Developed as a proof-of-concept for the Desktop Display featured above
  • Strengthened my knowledge of APIs and object-oriented programming skills
  • Useful on a second monitor, and developed before Spotify implemented their own miniplayer!

Memory Game

  • Memory-based, maze game where the player must follow a pre-determined path they are shown
  • Developed with an ELEGOO kit I acquired for a school project
  • C++ code generates a random route and multiple fail and win states independently

Guitar Hero

  • Modified and built an online design for guitars to play Guitar Hero and related games
  • Saved hundreds of dollars from retail units with USB microcontrollers and mechanical keyboard switches
  • Total cost of all purchased and 3D printed parts was about $25 per guitar
